The Column component takes all direct children and makes columns out of them.

      <Columns>
        <Card>Column 1</Card>
        <Card>Column 2</Card>
      </Columns>

Custom flexBasis (for narrower columns)

      <Columns flexBasis="0">
        <Card>Column 1</Card>
        <Card>Column 2</Card>
        <Card>Column 3</Card>
        <Card>Column 4</Card>
      </Columns>
